Languide provides high-level editing commands, such as turning an in-line block of code into a procedure, and substituting a call to the procedure for the original code. Languide is one of a pair of companion packages. The other one, Versor, provides flexible keybindings which can be used to navigate code in terms of its structure. Part of Versor's functionality is provided by the lower-level parts of Languide. See Versor and Languide (Versor: versatile cursors for GNUemacs), for more information. Most of Languide's functionality can be, and normally would be, accessed through Versor, but it is possible to use Languide commands directly. This manual describes the direct use of them; for use through Versor, See Top (versor).
