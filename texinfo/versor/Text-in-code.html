9.3.3.4 Text in code

Type: boolean Default value: t Whether versor should switch dimensions for string literals and comments. This requires font-lock-mode to be used. The current dimensions are then remembered separately, and switch to the last one used in that kind of place, as point moves between code, comments and strings. See versor-text-faces for the faces used to recognize this, building on font-lock-mode.
