5.2 Statement handling

This group of commands acts mostly on imperative statements.

  • Unify Statements: Unify Statements This makes the selection into a compound statement.
  • Make conditional: Make the selected code conditional
  • Make repeating: Make the selected code iterative
  • Remove control: Remove a control construct
  • Decision point: Move to enclosing decision point This moves point to the most nearly enclosing decision point, that is, a suitable place for putting a new condition.
