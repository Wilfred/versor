7.3 Using voice input

Some parts of Versor are particularly suited to use with voice input, such as vr-mode http://emacs-vr-mode.sourceforge.net/. Some of the voice or menu commands read their arguments through a system built on top of completing-read, that is designed to allow fast selection of possibilities without needing typing. See Flexi-choose, for details of this.
