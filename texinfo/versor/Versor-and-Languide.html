8 Versor and Languide

Language provides “language guided editing”, guiding editing operations using the syntax of the file being edited. The statement-based dimensions of Versor use the lower levels of Languide, and the refactoring operations use the higher levels of it. Languide is a cross-language package, with definitions for several popular programming languages. See Languide and Versor (Languide: language guided editing), for languide's end of the story..
