7 Versor and languide

Language provides “language guided editing”, guiding editing operations using the syntax of the file being edited. The statement-based dimensions of versor use the lower levels of languide. Languide is a cross-language package, with definitions for several popular programming languages. See Languide and versor (Languide: language guided editing) for Languide's end of the story..
